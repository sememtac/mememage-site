Unsigned, so a direct download needs one manual unblock the first time — macOS: System Settings → Privacy & Security → Open Anyway; Windows: More info → Run anyway. The one-line installs below skip that step entirely.

One-line install · skips the security prompt

$ curl -fsSL https://mememage.art/install.sh | bash
PS> irm https://mememage.art/install.ps1 | iex

Downloads the right build, installs it, and launches it — no Gatekeeper or SmartScreen step, because a script download isn’t flagged the way a browser download is.
